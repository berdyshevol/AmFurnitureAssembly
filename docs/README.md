# AM Furniture Assembly

Website for a professional furniture assembly service based in Denton, TX. Built with Next.js, React, TypeScript, and Tailwind CSS. Includes a gallery, pricing, blog, testimonials, and contact form.

## Photos

All website photos live in **`public/photos/`**. This is the folder referenced throughout the site (gallery, services, hero banners) via `/photos/...` URLs.

> The root-level `photos/` folder is gitignored raw source material — not used by the app.

SVG icons and the logo are in **`public/`**.
